# C  Programs

This repository contains a comprehensive set of simple, human-friendly C programs covering all major topics for beginners. Each program is well-commented and designed to help you learn C programming step by step.

## Topics Covered

- Basic Syntax & Output
- Variables & Data Types
- Operators & Expressions
- Control Flow (if-else, switch, loops)
- Functions & Recursion
- Arrays, Strings, and Pointers
- Structures, Unions, Typedef
- File Handling (read/write/create)
- Memory Management
- Enums, Macros, Bitwise Operators
- Error Handling & Input Validation
- Real-life Examples (Simple Calculator, etc.)

## How to Use

1. Clone this repository:
   ```sh
   git clone https://github.com/SamarthKapdi/C.git
   ```
2. Open any `.c` file in your favorite C IDE or editor.
3. Compile and run using a C compiler, for example:
   ```sh
   gcc hello_world.c -o hello_world
   ./hello_world
   ```
4. Explore, modify, and learn!
